# Student-Portal-Site
Simple site for student portal made with PHP native with MVC design pattern

## Features of the website include

## 1) First student must login

![login](https://user-images.githubusercontent.com/47919868/105636725-50cfa400-5e72-11eb-833d-9c2626c4f0bd.PNG)

## 2) Each student has a profile where he can view his grades and his registered courses 
![profile](https://user-images.githubusercontent.com/47919868/105636726-52996780-5e72-11eb-86eb-602364b99673.PNG)

## 3) Students can register courses
![courses](https://user-images.githubusercontent.com/47919868/105636727-53ca9480-5e72-11eb-892b-9fd07953c6b7.PNG)

![success](https://user-images.githubusercontent.com/47919868/105636728-54fbc180-5e72-11eb-86ee-31270be38e7a.PNG)

## 4) The website will not allow students to register courses with conflicting schedule 
![conflict](https://user-images.githubusercontent.com/47919868/105636729-575e1b80-5e72-11eb-87a1-88f09a40d3fd.PNG)

## 5) Students can view their schedule
